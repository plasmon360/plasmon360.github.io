Update Gnuplot plot automatically

Published: Thu 07 January 2010
By juluribk

In Free Software.

tags: gnuplot graphs Linux

To update gnuplot plot automatically. Run the command [cc]gnuplot gnuplot_script.txt[/cc] in terminal. (note: for some unknown reason it does not work for me on cygwin xterm, however it works on linux terminals.)

gnuplot_script looks like this:

[cc lang="gnuplot"]
plot "data.dat" using 1:2 w lp
replot
pause -5
reread
[/cc]
